# My First Portfolio

This is my first personal portfolio website, built using **HTML, CSS, and JavaScript**. I used a template from [ThemeWagon - Luther](https://themewagon.com/themes/luther/) as the base and customized it to showcase my work and skills.

## Features

- **About**: A brief introduction about me.  
- **My Work**: A showcase of my projects and achievements.  
- **Contact**: A simple contact section to get in touch with me.  

## Style

The design is **clean, minimalistic, and easy to navigate**, focusing on simplicity and readability.

## Installation

1. Clone the repository from GitHub.  
2. Open `index.html` in your browser to view the portfolio.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)
